# Roadmap

## v0.1 — Project Foundation ✅

- GitHub repository
- Project documentation
- Basic project structure

## v0.2 — Research Coach MVP

Goal:

Help researchers manage long-term research projects.

Features:

- Accept a research goal
- Break it into milestones
- Generate daily tasks
- Track completion status

---

## v0.3 — Paper Revision Assistant

Features:

- Review paper structure
- Generate revision checklist
- Provide writing suggestions
- Track revision history

---

## v0.4 — Long-Term Memory

Features:

- Remember previous conversations
- Track research progress
- Build user profile
- Personalized coaching

---

## v0.5 — Academic Coach

Expand beyond research.

Support:

- Coursework
- Programming projects
- Exam preparation
- LeetCode practice
- Thesis writing

---

## v1.0 — GotYou

A complete AI Academic Coach that helps students achieve long-term academic goals through planning, memory, reflection, and personalized guidance.
